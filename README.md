初始化->填充文件夹/填写表格->填写模组信息->生成模组->复制生成的文件并上传

### 运行

两种启动方式可选，任选其一即可

#### [首选]通过可执行文件运行

双击`StellarisAdvisorAdvisor.exe`运行程序，如无意外，这是本文件夹内唯一的.exe程序文件

可能在您的电脑中会显示为`StellarisAdvisorAdvisor`不含`.exe`，这是正常的
